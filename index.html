How It Works

Three steps to playing Minecraft

No Python. No Java pre-install. No complicated setup. Everything is bundled.

1

Download the launcher

Download the correct file for your OS below. Python, Java, and all dependencies are bundled inside — no separate installs.

2

Run the setup wizard

On first launch, a friendly wizard walks you through picking your Minecraft directory and RAM. Takes under 60 seconds.

3

Play Minecraft

Hit the LAUNCH button. UltraWater downloads Minecraft if needed, applies FPS optimizations, and gets out of the way.

Requirements

What do I actually need?

Less than you think.

End Users (Playing)

  • Windows 10+, macOS 12+, or Linux x64
  • 4 GB RAM (8 GB recommended)
  • Java — auto-installed by wizard
  • A Minecraft account (for online servers)
  • ✓ Python NOT required
  • ✓ No pip installs needed

Developers (Building from Source)

  • Python 3.10+
  • pip install -r requirements.txt
  • pyinstaller (for building)
  • Inno Setup (Windows installer)
  • create-dmg (macOS DMG)

Minecraft Requirements

  • Java 17+ (for MC 1.17+)
  • Java 8+ (for older versions)
  • 2 GB VRAM recommended
  • 2 GB+ disk space per version
  • All auto-managed by UltraWater
